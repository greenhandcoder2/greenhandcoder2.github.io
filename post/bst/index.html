二叉搜索树

问题

有一个机场，并且有一个跑道用于飞机的降落，飞机降落的具体时间是 ttt，降落过程需要 kkk min， 我们在一个集合 ∣\mid∣ R ∣\mid∣ = n 中，如何在 O(lgn)O(lgn)O(lgn) 时间复杂度内确定飞机是否能降落呢？

问题转化

集合R是时间的集合，每一个元素都是一个时间，我们的目标是在 O(lgn)O(lgn)O(lgn) 时间复杂度内找到一个元素插入的点，并且比较前一个元素和后一个元素，判断能否插入。

例如：R = {37， 41.2， 49， 56.3}, k = 3

  1. 插入53， ok，因为53介于49和56.3，并且差值大于3
  2. 插入44， 不允许，因为44和41.2差值小于3
  3. 插入20 不允许，因为20是过去的时间

各种已知的数据结构的问题

普通数组(unsorted array/list)

未排序的数组所有的操作都消耗 O(n)O(n)O(n) 时间复杂度，不符合要求

排序数组(sorted array)

  1. 找到插入的位置 R[ i ]，可以使用binary search，消耗 O(lgn)O(lgn)O(lgn)
  2. 比较 R[ i - 1 ]和 R[ i + 1 ],消耗 O(1)O(1)O(1)
  3. 插入 O(n)O(n)O(n)，因为要右移所有插入之后的元素

链表(sorted list)

插入只需要 O(1)O(1)O(1)，但无法使用binary search

Heaps： max/min

因为我们插入实在末尾插入的，插入的元素只会在对应的分支进行比较然后找到适合的位置，不会对另一个分支进行查找
缺陷：查找 >= k / <= k的元素花费 O(n)O(n)O(n)，同理，dictionary也有这个缺陷。

Binary Search Tree

目前，我们已知的数据结构无法做到 Insert(x)Insert(x)Insert(x) 和查找上一个最小的元素以及下一个比它大的元素同时在 O(lgn)O(lgn)O(lgn)时间复杂度内完成，但如果 sorted array能够让Insertion操作在O(lgn)O(lgn)O(lgn)时间复杂度内完成，那么我们就可以达到目标。

特点

BST拥有一个结点值 node x，和一个结点的value key(x)key(x)key(x)，并且不同于Heap的地方在于它具有指向关系，pointer：parent(x)parent(x)parent(x), left(x)left(x)left(x), right(x)right(x)right(x)

约束条件

  1. 对于所有的node x，如果y是 left(x)left(x)left(x),那么 key(y)key(y)key(y) <= key(x)key(x)key(x)
  2. 同理，对于右子树，有key(x)key(x)key(x) <= key(y)key(y)key(y)
  3. 对于左右子树而言，也必须是BST，并且没有重复结点

BST的优点

  1. 对于插入，我们只需要不断比较取寻找左右分支，最后就可以找到插入的位置，花费时间为 O(h)O(h)O(h)，h为树的高度。
  2. 对于寻找小于t的值的数量，也只需要 O(h)O(h)O(h) 时间复杂度。
    1. 往下寻找到目标值
    2. 加上小于nodes的节点数
    3. 加上left分支

如何构建一颗BST

构建结点生成器

代码

class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

初始结点构建器

查找函数

  1. 通过root的value判断是否等于要查找的值，等于直接返回 True
  2. 如果root的value > key，往左子树查找，直到找到value或者找到叶节点返回False
  3. 同理，如果 value < key, 往右子树查找

代码

def search(root, key):
	if root.val == key:
		return True
	if root.val < key:
		if root.right is None:
			return False
		return search(root.right, key)
	return search(root.left, key) if root.left is not None else False

插入函数

  1. 如果插入的刚好时叶节点，直接返回创建的结点
  2. 如果插入的key < root.value 往左子树寻找插入位置
  3. 同理， 如果插入的 key > root.value，往右子树寻找插入位置
  4. 如果插入的 key == value，则不做修改，直接返回 root（不违背重复原则）

代码

def insert(root, key):
	if root is None:
		return Node(key)
	else:
		if root.val < key:
			root.right = insert(root.right, key)
		elif root.val == key:
			return root 
		else:
			root.left = insert(root.left, key)
	return root

获取最小值

往左子树不断的寻找，找到最左边的叶子结点即可

代码

def get_min(root):
	current = root
	while current.left is not None:
		current = current.left
	return current

获取最大值

同理，往右子树不断寻找，直到找到最右边的叶子结点

代码：

def get_max(root):
	current = root
	while current.right is not None:
		current = current.right
	return current

删除结点

1.删除结点没有左右孩子，直接删除即可

2. 删除结点右一个孩子，则将删除结点赋值为他的孩子结点，并且删除孩子

3. 删除的孩子右两个结点，则需要找到删除结点的下一个中序后继结点，然后将该节点的值替换为中序后继结点的值，最后删除中序后继结点

代码

def delete_node(root, key):
	if root is None:
		return root
	if root.val < key:
		root.right = delete_node(root.right, key)
	elif root.val > key:
		root.left = delete_node(root.left, key)
	else:
		#结点值和删除的结点值一样
		
		#根结点只有一个子结点或者没有子结点
		if root.left is None:
			temp = root.right
			root = None
			return temp
		elif root.right is None:
			temp = root.left
			root = None
			return temp
		#获取中序后继节点
		temp = get_min(root.right)
		#赋值
		root.val = temp.val
		#删除中序后继节点
		root.right = delete_node(root.right, temp.val)
	return root

流程：找到删除的结点 →\rightarrow→ 通过对右子树使用get_min（）找到下个中序后继结点 →\rightarrow→ 将要删除结点的值赋值为后继结点的值 →\rightarrow→ 递归的删除右子树的后继结点

删除优化

上面的删除过程我们使用递归去删除后继结点，这里时有可以优化的地方的，因为有两个孩子的 root 的后继只有两种情况：

  1. 后继结点是右子树的最左叶子
  2. 后继结点是右孩子

对于这样的情况，我们可以通过追迹后继结点的 parent 结点，当找到最左叶子时，我们同时保留了它的 parent的信息，而这个最左叶子就是我们的后继，将值赋给 root 之后，我们将 parent 的左子树设置为空结点，删除succ 的目的也就达到了。

代码

def delete_node(root, key):
	if root is None:
		return root
	if root.val < key:
		root.right = delete_node(root.right, key)
		# 有else分支，最后一定返回完整的root，这里没必要返回
		#return root
	elif root.val > key:
		root.left = delete_node(root.left, key)
		#return root
	else:
		#两个孩子都为空
		# if root.left and root.right is None:
		# 	return None
		#有一个孩子为空
		if root.left is None:	
			temp = root.right
			root = None
			return temp
		elif root.left is None:
			temp = root.left
			root = None
			return temp
		#有两个孩子的情况
		succ_parent = root
		#查找后继结点
		succ = root.right
		while succ.left is not None:
			succ_parent = succ
			succ = succ_parent.left
		if succ_parent != root:
			succ_parent.left = succ.right
		else:
			succ_parent.right = succ.right
		root.val = succ.val
	return root

删除结点的时间复杂度： 最坏情况我们可能要遍历到最深的叶子结点，所以是 O(h)O(h)O(h) 。当我们BST为一个链表时，时间复杂度最坏为 O(n)O(n)O(n) 。

遍历

preorder(先序遍历)

先遍历根结点，再遍历左子树，最后遍历右子树
代码

def preorder(root):
	if root:
		#首先打印接结点值
		print(root.val)
		#递归左孩子
		preorder(root.left)
		#最后递归右孩子
		preorder(root.right)

inorder(中序遍历)

中序遍历先遍历左子树，再遍历根结点，最后遍历右子树
代码

def inorder(root):
	if root:
		#首先递归左孩子
		inorder(root.left)
		#打印接结点值
		print(root.val)
		#最后递归右孩子
		inorder(root.right)

postorder(后序遍历)

先遍历左子树，再遍历右子树，最后遍历根结点
代码

def postorder(root):
	if root:
		#首先递归左子树
		postorder(root.left)
		然后递归右子树
		postorder(root.right)
		最后打印根结点值
		print(root.val)

class Node:
    def __init__(self, key):
        self.left = None
        self.right = None
        self.val = key

def search(root, key):
	if root.val == key:
		return True
	if root.val < key:
		if root.right is None:
			return False
		return search(root.right, key)
	return search(root.left, key) if root.left is not None else False

def insert(root, key):
	if root is None:
		return Node(key)
	else:
		#根结点的值小于key，将右孩子做为根继续插入
		if root.val < key:
			root.right = insert(root.right, key)
		#根结点的值等于key，直接返回，不修改BST
		elif root.val == key:
			return root 
		#根结点的值大于key，将左孩子做为根继续插入
		else:
			root.left = insert(root.left, key)
	return root 

def get_min(root):
	current = root
	while current.left is not None:
		current = current.left
	return current

def get_max(root):
	current = root
	while current.right is not None:
		current = current.right
	return current

def delete_node(root, key):
	if root is None:
		return root
	if root.val < key:
		root.right = delete_node(root.right, key)
		return root
	elif root.val > key:
		root.left = delete_node(root.left, key)
		return root
	else:
		#两个孩子都为空
		# if root.left and root.right is None:
		# 	return None
		#有一个孩子为空
		if root.left is None:	
			temp = root.right
			root = None
			return temp
		elif root.left is None:
			temp = root.left
			root = None
			return temp
		succ_parent = root
		succ = root.right
		while succ.left is not None:
			succ_parent = succ
			succ = succ_parent.left
		if succ_parent != root:
			succ_parent.left = succ.right
		else:
			succ_parent = succ.right
		root.val = succ.val
	return root

def inorder(root):
	if root:
		#首先递归左孩子
		inorder(root.left)
		#打印接结点值
		print(root.val)
		#最后递归右孩子
		inorder(root.right)

def preorder(root):
	if root:
		print(root.val)
		preorder(root.left)
		preorder(root.right)

def postorder(root):
	if root:
		postorder(root.left)
		postorder(root.right)
		print(root.val)
