二分查找

原理：二分查找适用于f(x)f(x)f(x)为单调函数的序列（即已经排序好的数组），通过边界变换每次缩短一半的查找范围，使得比起顺序查找，时间复杂度降低为 O(lg(n))O(lg(n))O(lg(n))

题目1 基本二分查找

给定一个 n 个元素有序的（升序）整型数组 nums 和一个目标值 target ，写一个函数搜索 nums 中的 target，如果目标值存在返回下标，否则返回 -1。

示例1：

输入: nums = [-1,0,3,5,9,12], target = 9     
输出: 4       
解释: 9 出现在 nums 中并且下标为 4

示例2：

输入: nums = [-1,0,3,5,9,12], target = 2
输出: -1
解释: 2 不存在 nums 中因此返回 -1

代码：

class Solution:
    def search(self, nums: List[int], target: int) -> int:
        l, r = 0, len(nums) - 1
        while l <= r:
            mid = (l + r) // 2
            print(mid)
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                l = mid + 1
            else:
                r = mid - 1
        return -1

题目解释： 没有任何变形的二分法，只要注意边界l， r和不存在的情况即可

题目2搜索插入位置

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 O(log n) 的算法
示例1：

输入: nums = [1,3,5,6], target = 5
输出: 2

示例 2:

输入: nums = [1,3,5,6], target = 7
输出: 4

代码

class Solution:
    def searchInsert(self, nums: List[int], target: int) -> int:
        l, r = 0, len(nums) - 1
        while l <= r:
            mid = (l + r) // 2
            if nums[mid] == target:
                return mid
            elif nums[mid] < target:
                l = mid + 1
            else:
                r = mid - 1
        return l

思路：跟普通二分没有区别，唯一要注意的是如果没有找到要返回插入的位置，由于如果元素插在索引为0的地方，那么说明元素比数组的元素都小，此时 l=0 不会移动，r往左移动，这时候返回l就可以了。
当元素不插在0位置的时候，由于最后一定会执行一次 l = mid + 1，所以l在插入的位置。
所以直接返回 l 即可。

题目3 在排序数组中查找元素的第一个和最后一个位置

给定一个按照升序排列的整数数组 nums，和一个目标值 target。找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 target，返回 [-1, -1]。

进阶：
**你可以设计并实现时间复杂度为 O(log n) 的算法解决此问题吗？**示例 1：
示例1：

输入：nums = [5,7,7,8,8,10], target = 8
输出：[3,4]

示例 2：

输入：nums = [5,7,7,8,8,10], target = 6
输出：[-1,-1]

示例 3：

输入：nums = [], target = 0
输出：[-1,-1]

思路：题目描述时间复杂度为 O(log(n))O(log(n))O(log(n))，则想到要用二分查找，但是二分查找只能找到一个index，并不能保证左右边界会被索引到。
此时，我们可以考虑不断的二分去查找边界，然后把每次的值作为 flag 设置起来就可以找到左右边界了。

代码

class Solution:
    def searchRange(self, nums: List[int], target: int) -> List[int]:
        if len(nums) == 0:
            return [-1, -1]
        
        res1, res2 = -1, -1
        #左边界
        ls, rs = 0, len(nums) - 1
        while ls <= rs:
            mid = (ls + rs) // 2
            if nums[mid] < target:
                ls = mid + 1
            elif nums[mid] > target:
                rs = mid - 1
            else:
                res1 = mid
                rs = mid - 1
            
        #右边界
        le, re = 0, len(nums) - 1
        while le <= re:
            mid = (le + re) // 2
            if nums[mid] < target:
                le = mid + 1
            elif nums[mid] > target:
                re = mid - 1
            else:
                res2 = mid
                le = mid + 1
        
        return [res1, res2]

题目4求平方根

给你一个非负整数 x ，计算并返回 x 的 算术平方根 。

由于返回类型是整数，结果只保留 整数部分 ，小数部分将被 舍去 。

注意：不允许使用任何内置指数函数和算符，例如 pow(x, 0.5) 或者 x ** 0.5 。
示例 1：

输入：x = 4
输出：2

示例 2：

输入：x = 8
输出：2
解释：8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。

思路： 由于题目限定不能用内置函数，那么可以用牛顿迭代
牛顿迭代法就是找一个 (x0,f(x0))(x0,f(x0))(x0,f(x0)) 做切线，切线与 $ x$ 轴的交点 x1x1x1 ，过(x1,f(x1))(x1, f(x1))(x1,f(x1))再做切线，得到与 $ x$ 轴的交点 x2x2x2， 随着不断渐近，值会越来越逼近方程的真实根。
我们求a的平方根就是求方程f(x)−a=0f(x) - a = 0f(x)−a=0的解，再由地推公式简化得到根next=1/2(now+a/now)next = 1 / 2(now + a / now)next=1/2(now+a/now),我们a0a0a0取x0=1x0 = 1x0=1得到基本根，再设置循环跳出条件即可.

代码

class Solution:
    def mySqrt(self, x: int) -> int:
        if x == 0:
            return 0

        #初始值取a0 = 1
        next = (1 + x) / 2
        while True:
            next = 1 / 2 * (next + x / next)
            if abs(next ** 2 - x) < 1e-2:
                return int(next)
                break

注意最后要返回整数，要加int函数

移除元素

移除元素链接

给你一个数组 nums 和一个值 val，你需要 原地 移除所有数值等于 val 的元素，并返回移除后数组的新长度。

不要使用额外的数组空间，你必须仅使用 O(1) 额外空间并 原地 修改输入数组。

元素的顺序可以改变。你不需要考虑数组中超出新长度后面的元素。

示例 1：

输入：nums = [3,2,2,3], val = 3
输出：2, nums = [2,2]
解释：函数应该返回新的长度 2, 并且 nums 中的前两个元素均为 2。你不需要考虑数组中超出新长度后面的元素。例如，函数返回的新长度为 2 ，而 nums = [2,2,3,3] 或 nums = [2,2,0,0]，也会被视作正确答案。

示例 2：

输入：nums = [0,1,2,2,3,0,4,2], val = 2
输出：5, nums = [0,1,4,0,3]
解释：函数应该返回新的长度 5, 并且 nums 中的前五个元素为 0, 1, 3, 0, 4。注意这五个元素可为任意顺序。你不需要考虑数组中超出新长度后面的元素。

思路：题目要求在数组上进行原地修改而不使用额外的空间，我们立刻想到同为in place的快排，利用快排的划分。
将目标元素作为pivot然后两个指针分别指向首位，当满足条件时交换i，j的值，将数组分成两个部分，前面以i结束，均是和目标不同的数组，后部分以j为开端为value组成的数组。

由于题目没有限制数组的返回条件，直接返回第一个value的index(也就是i)就可以了

代码

class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        if len(nums) == 0:
            return 0
            
        i, j = 0, len(nums) - 1
        while i <= j:
            if nums[i] == val:
                if nums[j] == val:
                    j -= 1
                    continue
                else:
                    nums[i], nums[j] = nums[j], nums[i]
                    j -= 1
            i += 1
        return i

删除有序数组的重复项链接

给你一个有序数组 nums ，请你 原地 删除重复出现的元素，使每个元素 只出现一次 ，返回删除后数组的新长度。

不要使用额外的数组空间，你必须在 原地 修改输入数组 并在使用 O(1) 额外空间的条件下完成。

示例 1：

输入：nums = [1,1,2]
输出：2, nums = [1,2]
解释：函数应该返回新的长度 2 ，并且原数组 nums 的前两个元素被修改为 1, 2 。不需要考虑数组中超出新长度后面的元素。

示例 2：

输入：nums = [0,0,1,1,1,2,2,3,3,4]
输出：5, nums = [0,1,2,3,4]
解释：函数应该返回新的长度 5 ， 并且原数组 nums 的前五个元素被修改为 0, 1, 2, 3, 4 。不需要考虑数组中超出新长度后面的元素。

思路：注意时排序好的数组，这样我们直接用两个指针，一个指针指向当前的最大元素，另一个指针则遍历整个数组，当出现比当前元素大的情况时，i指针向前移动一位并赋值，当遍历完整个数组之后，i就是所有不重复元素的下标。注意返回长度 $ i + 1$

代码

class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        if len(nums) <= 1:
            return len(nums)
        
        i, j = 0, 1
        while j < len(nums):
            if nums[j] > nums[i]:
                nums[i + 1] = nums[j]
                i += 1
                j += 1
            else:
                j += 1
        return i + 1

移动零链接

给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

示例:

输入: [0,1,0,3,12]
输出: [1,3,12,0,0]

说明:
1. 必须在原数组上操作，不能拷贝额外的数组。
2. 尽量减少操作次数。

思路：同样in place的排序还是想到快排的划分，设置两个指针i，j分别指向最后一个不为0
的元素位置和0后面的第一个位置，遍历数组依次交换即可。

代码

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        i, j = -1, 0
        while j < len(nums):
            if nums[j] == 0:
                j += 1
                continue
            else:
                nums[i + 1], nums[j] = nums[j], nums[i + 1]
                i += 1
                j += 1

有序数组的平方

给你一个按 非递减顺序 排序的整数数组 nums，返回 每个数字的平方 组成的新数组，要求也按 非递减顺序 排序。

示例 1：

输入：nums = [-4,-1,0,3,10]
输出：[0,1,9,16,100]
解释：平方后，数组变为 [16,1,0,9,100]
排序后，数组变为 [0,1,9,16,100]

示例 2：

输入：nums = [-7,-3,2,3,11]
输出：[4,9,9,49,121]

思路：由于复数平方可能变大，所以不能直接平方，但数组本身时递增的，而平方之后非负部分和负数部分也是递增的，所以考虑分成两部分来排序。设立头尾指针i，j遍历数组然后依顺序添加到新的空白数组，最后返回翻转的结果

注意：
由于python reverse是in place的操作，所以不会返回任何值，用slice方式返回结果

代码：

class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        i, j = 0 ,len(nums) - 1
        res = []
        while i <= j:
            if nums[i] ** 2 > nums[j] ** 2:
                res.append(nums[i] ** 2)
                i += 1
            else:
                res.append(nums[j] ** 2)
                j -= 1
        return res[::-1]

两种反转python list的方法

my_list = [1, 2, 3, 4, 5]

my_list = [1, 2, 3, 4, 5]
reverse_list = my_list[::-1]
print(reverse_list)

长度最小的子数组

题目链接

给定一个含有 n 个正整数的数组和一个正整数 target 。

找出该数组中满足其和 ≥ target 的长度最小的 连续子数组 [numsl, numsl+1, ..., numsr-1, numsr] ，并返回其长度。如果不存在符合条件的子数组，返回 0 。

示例 1：

输入：target = 7, nums = [2,3,1,2,4,3]
输出：2
解释：子数组 [4,3] 是该条件下的长度最小的子数组。

示例 2：

输入：target = 4, nums = [1,4,4]
输出：1

示例 3：

输入：target = 11, nums = [1,1,1,1,1,1,1,1]
输出：0

思路：长度最小的子数组

  1. 可以使用暴力解法，遍历两次数组，第一层循环决定起始位置，第二层循环以第一层循环为起点遍历整个数组。时间复杂度
    (n+n−1+n−2+...+1)=O(n2)(n + n - 1 + n - 2 + ... + 1) = O(n^2)(n+n−1+n−2+...+1)=O(n2)
  2. 使用滑动窗体，当找到一个子数组 >= Target的时候记录子数组的长度，然后右移起始点继续进行比较，直到不满足大于等于target，然后移动终点进行循环。时间复杂度：遍历整个数组 O(n)O(n)O(n),，每次对起点的移动都是常数次，所以总时间复杂度O(n)O(n)O(n)

滑动窗体代码

class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        i, j = 0, 0
        _sum = 0
        res = float('inf')
        for j in range(len(nums)):
            _sum += nums[j]
            while _sum >= target:
                res = min(res, j - i + 1)
                _sum -= nums[i]
                i += 1
        return res if res != float('inf') else 0

暴力解法(leetcode 上python暴力解法不能通过，c++可以)

class Solution:
    def minSubArrayLen(self, target: int, nums: List[int]) -> int:
        res = float('inf')
        for i in range(len(nums)):
            _sum = 0
            for j in range(i, len(nums)):
                _sum += nums[j]
                if _sum >= target:
                    res = min(res, j - i + 1)
        return res if res != float('inf') else 0

最小覆盖字串

题目
给你一个字符串 s 、一个字符串 t 。返回 s 中涵盖 t 所有字符的最小子串。如果 s 中不存在涵盖 t 所有字符的子串，则返回空字符串 "" 。

注意：

  1. 对于 t 中重复字符，我们寻找的子字符串中该字符数量必须不少于 t 中该字符数量。
  2. 如果 s 中存在这样的子串，我们保证它是唯一的答案。

示例 1：

输入：s = "ADOBECODEBANC", t = "ABC"
输出："BANC"

示例 2：

输入：s = "a", t = "a"
输出："a"

示例 3:

输入: s = "a", t = "aa"
输出: ""
解释: t 中两个字符 'a' 均应包含在 s 的子串中，
因此没有符合条件的子字符串，返回空字符串。

思路：两种解法，一种是暴力循环，i 遍历整个数组，j 以 i 为起始位置向后遍历，当 j 到达末尾或者 t 中的元素全部找到，停止循环进入下一次查找

暴力查找代码（超出时间限制，不能AC）

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        d = dict()
        l = float('inf')
        for t in t:
            if d.get(t) is not None:
                d[t] += 1
            else:
                d[t] = 1
        for i in range(len(s)):
            temp = d.copy()
            j = i
            while j < len(s) and len(temp) != 0:
                if temp.get(s[j]) is not None:
                    if temp[s[j]] > 1:
                        val = temp[s[j]]
                        temp[s[j]] = val - 1
                        j += 1
                    else:
                        del temp[s[j]]
                        j += 1
                else:
                    j += 1
            if len(temp) == 0 and j - i < l:
                l = min(l, j - i)
                res = s[i : j]
        return res if l != float('inf') else ''

第二种方法是滑动窗口，i 依然作为起始，用 j 向后遍历，每当找到 t 中的所有元素时， i 向前移动到第一个属于 t 的元素并记录长度，然后 i 右移继续寻找 t 的字串。

图示：
第一步：找到包含 t 的子串

第二步：找到字串第一个属于 t 的字符

第三步：移动到下一格，继续循环

滑动窗口

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        need = collections.defaultdict(int)
        for c in t:
            need[c] += 1
        need_cnt = len(t)
        i = 0
        res = (0, float('inf'))
        for j, c in enumerate(s):
            if need[c] > 0:
                need_cnt -= 1
            need[c] -= 1
            if need_cnt == 0:
                while True:
                    if need[s[i]] == 0:
                        break
                    need[s[i]] += 1
                    i += 1
                if j - i < res[1] - res[0]:
                    res = (i, j)
                need[s[i]] += 1
                need_cnt += 1
                i += 1
        return s[res[0]: res[1] + 1] if res[1] != float('inf') else ''

螺旋矩阵

给你一个正整数 n ，生成一个包含 1 到 n2 所有元素，且元素按顺时针顺序螺旋排列的 n x n 正方形矩阵 matrix 。
示例 1：

输入：n = 3
输出：[[1,2,3],[8,9,4],[7,6,5]]

示例 2：

输入：n = 1
输出：[[1]]

思路：遵循左闭右开的原则， 从左到右 →\rightarrow→ 从上到下 →\rightarrow→ 从右到左 →\rightarrow→ 从下到上 进行遍历。

代码

class Solution:
    def generateMatrix(self, n: int) -> List[List[int]]:
        #构建初始矩阵
        matrix = [[0] * n for _ in range(n)]

        left, right, up, down = 0, n - 1, 0, n -1
        number = 1 #插入的数字

        while left < right and up < down:

            # 从左到右填充上边
            for x in range(left, right):
                matrix[up][x] = number
                number += 1
            
            #从上到下填充右边
            for y in range(up, down):
                matrix[y][right] = number
                number += 1

            #从右到左填充下边
            for x in range(right, left, -1):
                matrix[down][x] = number
                number += 1

            #从下到上填充左边
            for y in range(down, up, -1):
                matrix[y][left] = number
                number += 1

            # 缩小填充范围
            left += 1
            right -= 1
            up += 1
            down -= 1
        
        # 如果时奇数，填充中间
        if n % 2:
            matrix[n // 2][n // 2] = number
        
        return matrix
